# Welcome to the AgriPulse FMIS Wiki

## Overview

AgriPulse FMIS is a Farm Management Information System designed for sovereign edge computing. It runs on a 3-node Raspberry Pi 5 cluster with Hailo-10H AI acceleration, providing real-time sensor monitoring, AI-powered agronomy advice, and autonomous actuator control — all without any cloud dependency.
